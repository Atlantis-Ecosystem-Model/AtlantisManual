# Atlantis User Guide

This repository contains the Atlantis User Guide documentation, which is [rendered at this link](https://atlantis-ecosystem-model.github.io/AtlantisManual/) at every new commit.

## Status

The linked website contains the complete Atlantis User Guide updated as of September 2025. We have striven to format everything correctly but there may be still some visual glitches and oddities. If you encounter issues with the website, open an issue on this repo and we will address issues as they come up.

Note that the User Guide website contains links to the Wiki. As the Atlantis Wiki no is no longer hosted on the Confluence CSIRO portal, those links go nowhere. We are in the process of porting the old Wiki to a website, stay tuned.
